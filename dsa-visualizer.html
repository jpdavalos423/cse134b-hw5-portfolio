About the Project

The Data Structures & Algorithms Visualizer is an interactive tool built to help students understand how popular algorithms operate internally. The project animates each algorithm step-by-step, translating abstract logic into intuitive visual sequences.

I created the visualizer to support computer science learners and to deepen my own understanding of algorithmic behavior, recursion, and performance.

Key Features

  • Animations for sorting algorithms such as Merge Sort and Bubble Sort
  • Step, pause, and speed controls for interactive playback
  • Visual highlighting of comparisons, swaps, and recursive calls
  • Built with React for UI management and D3.js for rendering
  • Educational interface designed for clarity and engagement

Development Process

The project started with designing a system to convert algorithm operations into discrete visualization states. React handled UI components, while D3 managed the animated SVG elements. User testing helped determine ideal pacing, color choices, and control layout.

This project strengthened my ability to combine algorithmic reasoning with front-end engineering patterns.

Technical Implementation

Each algorithm is transformed into a sequence of recorded states, which D3 animates through transitions. React manages user controls and state changes, while D3 performs efficient DOM manipulation and rendering.

This separation ensures smooth animations and clean component logic.
