About the Project

Now Playing Display is a lightweight music information interface built for Raspberry Pi devices. It retrieves real-time Spotify playback data and presents it on a dedicated display, functioning like a compact smart music dashboard.

The project combines embedded hardware, API communication, and minimalist UI design to create an elegant and functional display.

Key Features

  • Real-time Spotify playback data using the Spotify Web API
  • OAuth authentication system with token refresh handling
  • Optimized layout for small or low-power screens
  • Backend logic built with Python and Flask
  • Clean and minimalist UI focused on readability

Development Process

Development began with testing OAuth flows and creating a stable polling mechanism for updating track metadata. Several UI prototypes were explored before landing on a minimal layout optimized for distance visibility.

Flask serves dynamically-rendered templates, while CSS ensures proper scaling across various screen sizes.

Technical Implementation

The display queries Spotify at set intervals, parses JSON metadata, and updates album art, track name, and artist information. Responsive design techniques allow the interface to adapt smoothly to a Raspberry Pi display.

Caching and efficient update logic reduce API usage and improve performance.

Challenges & Solutions

Managing OAuth refresh tokens was a major challenge due to strict expiration rules. Implementing automatic token renewal and error recovery ensured the system remained stable during extended playback sessions.

Future Enhancements

Planned improvements include animated visualizers, multi-platform support, and additional metrics such as listening history or track popularity.
